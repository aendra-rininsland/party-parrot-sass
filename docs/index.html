PARTY-PARROT-SASS

MAKE EVERYTHING THIS COLOURFUL

What is it?

party-parrot-sass is a Sass mixin for making everything in your life straight-hype.

Usage

  1. Install via NPM or Bower: 
    $ npm install party-parrot-sass --save
    $ bower install party-parrot-sass --save
  2. Import into your project: 
    @import 'party-parrot-sass/main';
  3. Include the mixin: 
    @include parrot(
  $prop: background-color,
  $speed: .5s,
  $direction: normal
);
